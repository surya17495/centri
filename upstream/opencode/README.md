# Upstream OpenCode reference

This directory holds the original [OpenCode](https://github.com/anomalyco/opencode)
README (`README.opencode.md`) and its community translations, plus the upstream download
stats (`STATS.md`), preserved verbatim from the upstream MIT-licensed project for
attribution.

These documents describe **upstream OpenCode, not Centri**, and may lag this fork. They
are kept here so the fork's upstream provenance stays visible and git history is
preserved. The authoritative README for this project is
[`../../README.md`](../../README.md); see [`../../FORK-NOTES.md`](../../FORK-NOTES.md)
for what Centri changed.

> The inherited OpenCode CI workflows (`stats`, `docs-locale-sync`) are inert on this
> fork — `stats` is gated to `github.repository == 'anomalyco/opencode'` and
> `docs-locale-sync` is disabled (`if: false`) — so they will not regenerate or overwrite
> anything here.
